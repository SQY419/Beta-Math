算pi
```
a=1
b=2√(0.5)
t=0.25
p=1
cf(10)
c=(a+b)/2
d=2√(a*b)
t=t-p*(a-c)^2
p=2*p
a=c
b=d
end
f=((a+b)^2)/(4*t)
```
f即为pi的值
